# Layer 1 вЂ” Project Health Score

Evaluates project execution quality using objective project metrics.

Components:

| Metric | Weight |
|---------|--------|
| Goal Progress | 40% |
| Completion Rate | 25% |
| Efficiency | 20% |
| Delay Penalty | 15% |

Produces:

- Overall Health Score
- Goal Progress Details
- Delayed Tasks
- Score Breakdown

---

# Layer 2 вЂ” Requirement Alignment

Measures how well the implementation aligns with the original client requirements.

Unlike keyword matching, this layer uses **semantic embeddings** generated by **Google Gemini**.

Process:

1. Embed every requirement.
2. Embed every project task.
3. Compute cosine similarity.
4. Match each requirement with the most relevant tasks.
5. Calculate weighted alignment score.
6. Detect orphan work and misalignment.

Outputs:

- Alignment Score
- Requirement Details
- Alignment Alerts
- Orphan Tasks

---

# Layer 3 вЂ” AI Judge

The AI Judge performs an expert-level project review using **Google Gemini**.

The model receives:

- Relevant Requirements
- Relevant Tasks
- Relevant Deliverables
- Health Metrics
- Alignment Results
- Delayed Tasks
- Goal Progress

It evaluates the project across five dimensions:

1. Requirement Coverage
2. Semantic Completeness
3. Execution Quality
4. Risk Detection
5. Consistency Across the Project

The AI returns:

- AI Judge Score
- Confidence
- Risk Level
- Executive Summary
- Key Issues
- Recommendations
- Missing Requirements

---

# Semantic Context Builder

Before calling Gemini, the system constructs a semantic context.

Instead of sending the whole project, it selects the most relevant information using embedding similarity.

This improves:

- Context quality
- Token efficiency
- AI reasoning

---

# Drift Detection

Detects when project execution starts drifting away from client requirements.

Signals include:

- Semantic Drift
- Missing Requirement Coverage
- Orphan Tasks

---

# Predictive Risk Forecast

Predicts future project risk using current project state.

Forecast includes:

- Misalignment Risk
- Delay Risk
- Failure Risk
- Overall Risk Level
- Risk Drivers

# API Endpoints

## Health Score

```
POST /project/{project_id}/health
```

Returns:

- Health Score
- Goal Progress
- Delayed Tasks

---

## Alignment Score

```
POST /project/{project_id}/alignment
```

Returns:

- Alignment Score
- Requirement Analysis
- Alerts

---

## AI Judge

```
POST /project/{project_id}/ai-judge
```

Runs:

- Semantic Context Builder
- Gemini AI Judge

Returns:

- AI Judge Score
- Recommendations
- Risk Assessment

---

## Full Evaluation

```
POST /project/{project_id}/evaluate
```

- Health

- Drift Detection

# Installation

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the server

```bash
uvicorn app.main:app --reload
```

Open Swagger

```
http://localhost:8000/docs
```
